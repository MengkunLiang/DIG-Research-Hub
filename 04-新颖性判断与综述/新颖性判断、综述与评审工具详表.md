# 新颖性判断、综述与评审工具详表

## 四、常见使用顺序

1. 先用 `OpenReview MCP / OpenAlex MCP / Paper Search MCP` 建立相关工作和评审语境。
2. 再用 `paper-glance-skill / literature-review / scientific-critical-thinking` 对核心论文做统一拆解。
3. 用 `ReviewAdvisor / NLPeer / AI Peer Review` 补 reviewer-style critique。
4. 用 `llm4novelty / HypoGeniC / HypoSpace` 从 novelty 与 hypothesis 角度再做一次判断。
5. 最后把结论沉到 [03-知识管理与证据沉淀](../03-知识管理与证据沉淀/README.md) 或 [06-写作、审稿与投稿](../06-写作、审稿与投稿/README.md)。
