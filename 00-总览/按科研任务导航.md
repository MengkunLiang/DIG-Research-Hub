1. 先进入对应任务页，看这一类问题常用的 `Skills / MCP / Agent / 数据源 / Benchmark` 组合。
2. 再沿着任务页中的内链进入详细类型页查全。
3. 若需要对某一类工具做系统扫库，再转到 [按资源类型导航](./按资源类型导航.md) 和 [98-全局索引](../98-全局索引/README.md)。
