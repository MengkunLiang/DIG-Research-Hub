## 5. 一条更稳的投稿链路

1. `PaperQA2 / citation-management` 固定内容证据链。
2. `OpenDraft / ScholarCopilot / Open Research` 组织正文、response 和 supplementary。
3. `official template / Auto-Resubmit` 处理模板与 venue 约束。
4. `Tectonic / latexmk / arxiv-latex-cleaner / venue-specific checker` 做 preflight。
5. `OpenReview / CMT / Editorial Manager / ScholarOne / arXiv` 完成最终提交。

## 6. 哪些节点仍然需要人工把关

- 结论定稿
- novelty 表述
- reviewer concern 的最终回应
- venue 与 journal 选择
- 最终提交包检查
