# 补库规则与待补清单

这一页用于把后续补库工作变成可持续、可分批推进的维护流程，而不是每轮都从零开始。

## 一、补库规则

1. 先判断主归属页，再补索引页和任务页。
2. 一个外部资源只保留一个主归属页做较完整说明。
3. 外部合集优先拆成“高价值入口 + 中文摘要 + 内链”，不直接镜像整页。
4. 新增条目尽量遵循 [统一收录模板](./统一收录模板.md)。
5. 每轮补库后同步更新 [更新日志](./更新日志.md)。
