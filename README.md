## 仓库设计原则

- 主入口优先服务“直接使用”，因此先提供任务页，再提供类型页。
- 外部资源的详细说明尽量只写在一个主归属页，其他页面用简述加内链引用，减少重复。
- 仓库说明以中文为主，收录对象以全球公开资源为主，英文与中文链接都会纳入。
- 对 MCP、Skills、Agent、Prompt、数据源、Benchmark 的关系，优先在任务页里组合展示，在类型页里完整收录。
