# 资源地图

本仓库当前采用“双导航”结构：

- 任务导航：围绕科研子问题，把 `Skills / MCP / Agent / Prompt / 数据源 / Benchmark` 组合到同一页。
- 类型导航：围绕资源类型，保留更完整的 Skills、MCP、文献、实验、投稿与专题页。

## 一、推荐使用路径

1. 先从 [按科研任务导航](./按科研任务导航.md) 或 [10-按科研子问题使用](../10-按科研子问题使用/README.md) 进入。
2. 再根据需要转到 [按资源类型导航](./按资源类型导航.md) 和各目录详细页查全。
3. 若需要快速查找某类工具，再使用 [98-全局索引](../98-全局索引/README.md)。

## 四、公共组织原则

- 任务页优先承担“组合展示”，类型页优先承担“完整收录”。
- 同一外部资源的长说明只保留在一个主归属页，其他页面用简述与内链引用。
- 仓库入口优先面向使用场景，而不是优先面向资源仓库名。
